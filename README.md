## NEXT-CHAT

A simple chat application created in Next.js with NextAuth.js and Ably.

Live demo hosted on Netlify: [visit here](https://next-chat.netlify.app)

### TECHNOLOGIES

- Framework: Next.js with Tailwindcss
- Authentication: NextAuth.js
- Database: Supabase (Postgresql)
- ORM: Prisma
- WebSocket: Ably
